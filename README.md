# ProEdgeBoost
专业-边缘加速|ProEdgeBoost (PEB) 专注于优化PC端游戏性能，提供稳定的网络连接和快速的游戏加载，确保玩家在复杂环境中也能享受流畅的游戏体验。

##技术栈
 - Tauri + Vue + TypeScript

## 初始化
```aiignore
yarn install
.\setup_aliyun_mirror.ps1
```
## 运行
```aiignore
yarn tauri dev
```

## 实现功能
- [x] 用户登录
- [x] 用户注册
- 客户端登录
- 客户端注册
- 客户端加速游戏时相关配置下发
- 服务器线路管理
- 服务器一键部署
- 服务器配置生成
- 服务器配置更改
- 服务器配置对接
